# ИНСТРУКЦИЯ ПО НАЧАЛЬНОЙ НАСТРОЙКЕ

Все перечисленные ниже справочники, константы, обработки и регистры доступны в подпистеме "Добавленные объекты".
 
## Общие настройки:

1.	Необходимо создать пользователей для всех сотрудников, которые будут работать с программой. Для этого в разделе «Администрирование» откройте справочник «Пользователи» и добавьте новых пользователей. Для каждого пользователя укажите полное имя и имя для входа, а на вкладке «Права доступа» выберите соответствующий профиль доступа.

## Подсистема «Обслуживание клиентов»:

1.	Необходимо создать группу и бота в Telegram для отправки уведомлений специалистам при создании документа «Обслуживание клиентов» или изменении даты/времени проведения работ или ФИО специалиста.
2.	Для работы Telegram-бота заполните константы:
«Идентификатор группы для оповещения» — укажите идентификатор группы Telegram; 
«Токен управления телеграм-ботом» — укажите токен бота. 
3.	Создать необходимую номенклатуру в справочнике «Номенклатура» для заполнения табличной части документа «Реализация товаров и услуг». Например, «Абонентское обслуживание» и «Работы специалиста».
4.	Заполнить константы «Номенклатура работы специалиста» и «Номенклатура абонентская плата» в соответствии с созданными элементами справочника «Номенклатура».
   
## Подсистема «Расчет зарплаты»:

1.	Необходимо создать сотрудников в справочнике «Сотрудники».
2.	Нужно заполнить график работы на требуемые периоды: для этого запустить обработку «Заполнение графика работы» и указать нужный период.
3.	Необходимо задать для сотрудников условия оплаты труда в регистре «Условия оплаты сотрудников».
